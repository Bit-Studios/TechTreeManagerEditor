Installation
  1. Install this mod using ckan, do not manually install
Quick start guide
  1. Creating your first tech tree
    1. Goto the tech tree Editor
    2. Once you have created your tech tree click the at the top of the page
      This will create a zip file with your tech tree
    3. Create a new folder at this path \Kerbal Space Program 2\BepInEx\plugins The folder name will be the name of your tech tree
    4. Extract the zip into the folder you created
    5. Edit the TTM config file \Kerbal Space Program 2\Config\TTM.config
      Change the value of TechTreeToUse to your tech trees name (folder name)

      Example for RealisticTechTree 
      
                            {
                                "TechTreeToUse": "RealisticTechTree"
                            }
    6. Boot your game and you should see your tech tree
